## Bug 2 — Illegal `router.replace()` Called During Render

**Type:** Console Error

**Message:**
> Cannot update a component (`Router`) while rendering a different component (`CheckoutClient`).

**File:** `app/[slug]/checkout/_common/CheckoutClient/CheckoutClient.tsx`

**Code Frame:**
```tsx
// Line 94–98
if (isEmpty) {
  router.replace(routes.storeCart()); // ← Line 96 — bad setState during render
  return null;
}
```

**Root Cause:**
`router.replace()` is being called directly inside the render function (not inside a `useEffect`), which triggers a state update on `Router` while `CheckoutClient` is still rendering.

## Bug 5 — Invalid HTML: Nested `<button>` Inside `<button>`

**Type:** Console Error

**Message:**
> In HTML, `<button>` cannot be a descendant of `<button>`. This will cause a hydration error.

### Nesting Chain:
```
SelectableCard          → renders <button>
  └── ImageUploader
        └── Button      → renders <button>   ← illegal nesting
```

### File Locations:

| Component | File |
|---|---|
| `SelectableCard` | `packages/ui/src/components/SelectableCard/SelectableCard.tsx:13` |
| `Button` | `packages/ui/src/components/Button/Button.tsx:20` |
| `ImageUploader` | `packages/ui/src/components/ImageUploader/ImageUploader.tsx:153` |
| `PaymentMethodSection` | `app/[slug]/checkout/_common/PaymentMethodSection/PaymentMethodSection.tsx:86` |
| `CheckoutClient` | `app/[slug]/checkout/_common/CheckoutClient/CheckoutClient.tsx:196` |

**SelectableCard code (root of nesting issue):**
```tsx
// SelectableCard.tsx:13
export const SelectableCard = ({ checked, onClick, children, layout = 'row', className }) =>
  <button onClick={onClick} className={cn(LAYOUT_CLASSES[layout], className)}>
    {children}  {/* ← contains another <button> via ImageUploader → Button */}
  </button>
```

**Button code:**
```tsx
// Button.tsx:20
return (
  <button  // ← nested inside SelectableCard's <button>
    disabled={disabled || loading}
    className={cn('relative rounded-xl font-ravi hover:cursor-pointer font-medium transition-opacity', ...)}
  >
```
